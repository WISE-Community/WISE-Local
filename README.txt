            ===================================================
                               Running WISE4
            ===================================================

Running WISE4 requires a Java Standard Edition Runtime 
Environment (JRE) version 6.0 or later.

=============================
Running With JRE 6.0 Or Later
=============================

(1) Download and Install a Java SE Runtime Environment (JRE)

(1.1) Download a Java SE Runtime Environment (JRE),
      release version 6.0 or later, from
      http://www.oracle.com/technetwork/java/javase/downloads/index.html

(1.2) Install the JRE according to the instructions included with the
      release.

      You may also use a full Java Development Kit (JDK) rather than just
      a JRE.

=============================
      STARTING WISE4
=============================

(1) Double-click on the "START WISE" icon.
	This will open up your Terminal application and starts up the WISE server. 
	Any logs will output in the Terminal window.

(2) Double-click on the "WISE Homepage" icon.
	This will open up your default browser and go to your local instance of WISE.

=============================
     PREVIEWING PROJECTS
=============================

(1) This instance of WISE comes with sample projects.
	Find them in the homepage and click on the "preview" link to preview.

=============================
      STOPPING WISE4
=============================

(1) Double-click on the "STOP WISE" icon.
	This will open up your Terminal application and shuts down the WISE server. 
	Any logs will output in the Terminal window.
